SQL Data Cleaning Project (MySQL)

## Project Overview

This project demonstrates practical SQL data cleaning techniques using MySQL. The objective was to transform a messy employee dataset into a clean, analysis-ready dataset.

## Dataset

Employee records containing inconsistencies such as:

- Extra spaces
- Mixed letter cases
- Invalid phone numbers
- Different date formats
- Currency symbols in salary
- Missing values
- Duplicate records

## Data Cleaning Tasks

- Trimmed unnecessary spaces
- Standardized text formatting
- Converted emails to lowercase
- Formatted Nigerian phone numbers
- Cleaned salary values
- Converted salary to DECIMAL
- Standardized hire dates
- Removed invalid records
- Removed duplicate records
- Created a clean employee table

## SQL Techniques Used

- UPDATE
- DELETE
- ALTER TABLE
- CREATE TABLE
- INSERT INTO
- TRIM()
- LOWER()
- UPPER()
- CONCAT()
- SUBSTRING()
- STR_TO_DATE()
- REGEXP_REPLACE()
- REGEXP
